# SintoniaFit - Body Transformation App

A full-stack application for meal planning and exercise routines with AI integration.

## Project Status

✅ GitHub Repository configured
✅ Vercel deployment set up
⏳ Application code migration in progress

## Deployment

This project is deployed on Vercel and will automatically deploy when code is pushed to the main branch.

**Live URL**: https://sintonia-fit.vercel.app

## Technology Stack

- **Frontend**: React + Vite + TypeScript
- **Backend**: Express.js + Node.js
- **Database**: PostgreSQL
- **AI Integration**: Gemini API
- **Deployment**: Vercel (Free tier)

## 24/7 Uptime

This application is configured to run 24/7 on Vercel's free tier, ensuring the app is always available without browser interaction required.
